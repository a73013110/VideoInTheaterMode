Video in Theater Mode

How to use

1. Open a video page, for example: youtube.

2. Although seldom the web page will set more than one video as invisible, ensure there is only one video.

3. Click this chrome extension button or use hotkey: Ctrl+Shift+F to turn the theater mode ON/OFF.

※ If no or more than one video in the page, this extension will not work and show an alert

※ There are existing some bugs in this extension with few influence, so Ctrl+R to refresh page when encountering
